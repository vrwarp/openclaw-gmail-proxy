3 · Give OpenClaw a token

Issue an agent credential on the Credentials page ({{ creds | selectattr("revoked","equalto",false) | list | length }} active), then register this proxy as an MCP server in ~/.openclaw/openclaw.json:

{
  "mcp": {
    "servers": {
      "gmail-proxy": {
        "url": "https://PROXY_HOST:{{ mcp_port }}/mcp",
        "transport": "streamable-http",
        "headers": { "Authorization": "Bearer ${GMAIL_PROXY_TOKEN}" }
      }
    }
  }
}

Replace PROXY_HOST with this host's VM-facing address and set GMAIL_PROXY_TOKEN to the issued credential.
